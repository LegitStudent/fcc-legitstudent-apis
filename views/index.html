API Usage

# Timestamp Microservice

API Endpoint: /timestamp

Possible Actions to Endpoint

GET request to /?time=[time string], where [time string] is a timestamp with a UNIX or Natural (MMMM DD, YYYY) format.

Expected response: A JSON object that display given time string in both UNIX and Natural time.

{ "unix": 1450137600, "natural": "December 15, 2015" }

# Request Header Parser Microservice

API Endpoint: /request

Possible Actions to Endpoint

GET request to /

Expected response: A JSON object that shows the IP address, language, and operating system of the browser/ application that made the request.

{"ipaddress":"127.0.0.1","language":"en-GB","software":"Windows NT 10.0; WOW64"}
